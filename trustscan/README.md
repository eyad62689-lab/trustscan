# TrustScan

أداة فحص آلي للثقة والخصوصية لأي موقع أو صفحة تطبيق (App Store / Google Play).

## البنية
- الواجهة: Astro (ثابتة) — `src/pages/index.astro`
- الفحص الفعلي: Netlify Function — `netlify/functions/analyze.js`

## النشر على Netlify
1. ارفع هذا المجلد إلى مستودع GitHub جديد.
2. من Netlify: **Add new site → Import from Git**، اختر المستودع.
3. إعدادات البناء تُقرأ تلقائياً من `netlify.toml` (build: `npm run build`, publish: `dist`, functions: `netlify/functions`) — لا حاجة لتعديل شي.
4. بعد أول نشر، جرّب من الموقع نفسه فحص رابط مثل `github.com` للتأكد إن الدالة تعمل.

## ملاحظة مهمة عن دقة الفحص
- الدالة تتصل مباشرة بالموقع المُدخل (HTTPS headers + شهادة TLS + محتوى الصفحة الرئيسية) وتحلّلها فعلياً وقت الطلب — ما فيه بيانات مزيفة أو تقديرية.
- فحص صفحات المتاجر (App Store / Google Play) يقتصر على ما ينشره المطوّر في الصفحة نفسها (رابط سياسة الخصوصية، ملصق الخصوصية) لأنه لا يوجد اتصال HTTP بالتطبيق نفسه نفحصه مباشرة.
- الأداة لا تخزّن أي رابط تم فحصه ولا تنشئ سجلاً — كل فحص يحدث لحظياً فقط.

## الخط
النص العربي في الواجهة يستخدم **Amiri** (خط نسخ كلاسيكي مفتوح المصدر) للعناوين والمحتوى الأساسي، مع **IBM Plex Sans Arabic** للنصوص الثانوية و**JetBrains Mono** للبيانات التقنية (الروابط، الأكواد). الخطوط تُحمّل من Google Fonts تلقائياً.
